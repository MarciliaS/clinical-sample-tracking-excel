Clinical Sample Tracking & Validation Tool
📌 Overview

This project simulates a clinical trial sample tracking workflow using Excel. It reconciles expected vs received samples, flags missing records, and validates study consistency.

Key Features
Sample reconciliation using XLOOKUP
Error handling with IFERROR
Data validation flags (IF)
Metrics using COUNTIFS
Study consistency validation using UNIQUE
Conditional formatting for visual QC review

Use Case
Relevant for:

Clinical Data Coordinator roles
Sample management workflows
Clinical trial operations
Data quality and validation tasks
